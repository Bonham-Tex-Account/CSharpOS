Before implementing any function that involves major design decisions, ask the user how they would like to approach it rather than deciding unilaterally.

**Why:** User wants to be in control of design choices and discover problems naturally.
**How to apply:** When about to implement a non-trivial function, identify the key design decisions first and ask before writing any code. [[project-csharpos]]
